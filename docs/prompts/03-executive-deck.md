# 03 — Executive deck

**Date:** 2026-04
**Asked for:** Turn the analysis into a board-ready executive summary deck (PowerPoint).

## Prompt(s) quoted
> Build me an executive summary deck I can hand to the board.

## What was produced
- First version of `analysis/build_deck.py` (python-pptx).
- Initial 12–15-slide deck with title, problem, peer set, SEDA deltas, curriculum cross-tab, recommendation.
- Established the 16:9 canvas, palette, and footer pattern that the style guide later codified.

## Why this mattered
Switching from a written report to a deck forced every claim to fit a single slide with a single number — which is what shook out the slides where I was hand-waving. It also created the artifact (the .pptx) that drives every later iteration: re-render, screenshot, fix, repeat.
